How to use this Phone Number Input Mask script? 1. Go to Settings > Page Settings > Custom Code 2. Copy this script code and paste on the "Body (footer) scripts" 3. Create a New page from Pages > Add New 4. Edit the page with "Bricks Builder" 5. Take a container > Add a From Element from left side 6. Take field for Phone number 7. The Phone number id should be "phone" 8. Save the page and visit a new window Thank you. If you need video tutorial you may check YouTube.
